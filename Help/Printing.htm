Printing

The display area may be printed on a single page.

  • Click the File menu and select Print Setup to select the printer, paper size, orientation, and other printer properties.
  • Click the File menu and select Print Preview to view the page as it would appear when printed. The display area is always scaled so that it will fit on one page.
  • Click the File menu and select Print to print the page. Set the image generation to coarse for low-resolution and faster printing. Set to fine for higher resolution and slower printing. (See note below.)

Note:

  • The resolution for image generation is independent of the resolution of the printer. Even if the printer is set to print at high resolution, the printed image will appear coarse if the image generation resolution is set to coarse.
