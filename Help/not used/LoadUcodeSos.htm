Loading UCODE-2005 Data

  1. Click the File menu and select New.
  2. If changes to the previous MV document have not been saved, a dialog box is displayed that allows you to click: Yes to save changes, or No to discard changes.
  3. The Model Selection dialog box is displayed. Select UCODE 2005 - SOS from the drop-list box, and then click OK.
  4. The UCODE Data File dialog box is displayed. Specify the _SOS file in the text box by clicking the corresponding Browse button and navigating to the file.
  5. Click OK.
  6. The Data Selection dialog box is displayed. There is only one choice in the drop-list box. It represents the sum of squared weighted residuals. Click OK.

After the model data are loaded, the opening display will be a bounding box viewed from overhead (from +z axis toward -z direction) at the selected time in Step 6 above. The first parameter in the _sos file is represented on the X-axis. The second parameter in the _sos file is represented on the Y-axis. The third parameter in the _sos file is represented on the Z-axis.

Scalar data are displayed by selecting solid or isosurface from the Show menu. Scalar values used to set the color bar as follows:

  • Red represents the maximum scalar value.
  • Blue represents the minimum scalar value.
  • However, if scalar values are uniform over the grid, then red represents s+1, and blue represents s-1, where s is the uniform scalar value.
