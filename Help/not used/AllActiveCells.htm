Assuming All Active Cells

  • The last item under the Action menu is the Advanced submenu, which contains the Assume All Active Cells command.
  • For finite-difference models (MODFLOW-2000, MODFLOW-2000 GWT, MODFLOW-96, MOC3D, and MT3DMS), if all cells in the model grid are active, invoking this command can speed up calculations required for graphical rendering.
  • However, if the model grid contains inactive cells, this command should not be invoked. Otherwise, inactive cells will be displayed as active.
  • For the finite-element model SUTRA, all cells are active, so this command is disabled.
