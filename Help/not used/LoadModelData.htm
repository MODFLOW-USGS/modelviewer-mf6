Loading Model Data

After a model run has been successfully executed, the model data may be loaded to Model Viewer for visualization. Please click on a model below to see the directions for loading data:

  • MODFLOW-2000 and MODFLOW-2005
  • MODFLOW-2000 GWT
  • MODFLOW-96
  • MOC3D (Version 3.5)
  • MT3DMS
  • PHAST
  • SUTRA (Versions 2.2, 2.1, and 2.0)
  • UCODE-2005

Note:

  • Model Viewer does not display pathlines by themselves. Instead, the pathline file may be loaded as an optional file along with results from MODFLOW-2000, MODFLOW-2000 GWT, MODFLOW-96 MOC3D, or MT3DMS.
