Pathlines Toolbox

The Pathline toolbox controls the display of pathlines.
  • If the Pathline toolbox is not visible, display it by clicking the Toolbox menu and selecting Pathline.
  • Clicking Default will assign default settings to the selected tab. Default settings are given in parentheses below.
  • The Appearance tab allows you to:
    • Select Line to show pathlines as lines.
    • Select Tube to show pathlines as narrow tubes.
    • Set the pathline tube diameter. (Model Viewer determines a default tube diameter of 1. To increase the diameter, specify a value greater than 1. The decrease the diameter, specify a value smaller than 1.)
  • The Color tab allows you to set the travel times at the red and blue endpoints of the color spectrum. Either red or blue may serve as the early time.

  • The Threshold tab allows you to:
    • Select No threshold to turn off thresholding.
    • Select Threshold by simulation time to show the pathline segments having travel times up to the time at the current display.
    • Select Threshold by time range to show only those pathline segments for which the travel time is within the mininum and maximum times specified in the text boxes.
    • Set the minimum and maximum times for the threshold range.
