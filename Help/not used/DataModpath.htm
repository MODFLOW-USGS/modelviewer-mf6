MODPATH Data Requirements

Date requirements for visualizing MODPATH results are as follows:

  • Pathlines should be saved as a standard or compact text file.

Notes:

  • Model Viewer does not display pathlines by themselves. Instead, the pathline file may be loaded as an optional file along with results from the following models: MODFLOW-2000, MODFLOW-2000 GWT, MODFLOW-96, MOC3D, and MT3DMS. The pathline file must be place in the same folder as the "name file" for the model.
  • For versions of MODPATH that do not specify whether forward or backwards tracking is used, the user must specify the direction when loading the file.
