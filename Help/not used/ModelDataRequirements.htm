Model Data Requirements

  • Before using Model Viewer, you must first run a model to generate simulation results.
  • Model input files should be prepared according to instructions in the model documentation or user's guide.
  • However, a few additional requirements are necessary to visualize results in Model Viewer. These additional requirements depend on the model:
    • MODFLOW-2000 and MODFLOW-2005
    • MODFLOW-2000 GWT
    • MODFLOW-96
    • MOC3D (Version 3.5)
    • MODPATH
    • MT3DMS
    • PHAST
    • SUTRA (Version 2.2, 2.1, and 2.0)
    • UCODE-2005
  • After running a model, if you anticipate moving model data files into another folder, you should read the discussion on moving model data files.
