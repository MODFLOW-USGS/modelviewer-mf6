Loading PHAST Data

  1. Click the File menu and select New.
  2. If changes to the previous MV document have not been saved, a dialog box is displayed that allows you to click: Yes to save changes, or No to discard changes.
  3. The Model Selection dialog box is displayed. Select Phast HDF from the drop-list box, and then click OK.
  4. The Phast Data File dialog box is displayed. Specify the HDF file in the text box by clicking the corresponding Browse button and navigating to the file.
  5. Click OK.
  6. The Data Selection dialog box is displayed. From the drop-list boxes, select the time and data type for the initial display. Then click OK.

After the model data are loaded, the opening display will be a bounding box viewed from overhead (from +z axis toward -z direction) at the selected time in Step 6 above.

Scalar data (for example, heads, concentrations, etc.) are displayed by selecting solid or isosurface from the Show menu. Scalar values at the selected time are used to set the color bar as follows:

  • Red represents the maximum scalar value.
  • Blue represents the minimum scalar value.
  • However, if scalar values are uniform over the grid, then red represents s+1, and blue represents s-1, where s is the uniform scalar value.

Vector data are displayed by selecting Vectors from the Show menu. If nodal velocity data are available, then the velocity vectors are scaled so that, for the selected time, the length of the vector with the highest magnitude is approximately one-tenth the diagonal distance across the bounding box.
